# BIO 40B — Blood (Hematology) Quiz

Interactive 50-question multiple-choice quiz on blood and hematology.

- **Immediate feedback** — the correct answer is revealed as soon as you pick.
- **Retake missed questions** — at the end, retake only the ones you got wrong, and keep retaking round after round until you get every question correct.
- **Two modes** — questions in order, or shuffled (question order *and* answer order randomized).
- **By-topic breakdown** — see which areas (RBCs, hemostasis, blood types, etc.) need more work.

Plain vanilla HTML/CSS/JS — no build step, no dependencies. Just open `index.html`.

Live site: https://producer456.github.io/bio40b-blood-quiz/
